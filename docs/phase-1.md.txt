### API Requirements

##### Phase 1


###### Section "Ingredients catalog"

> User can:
> 1. Add new ingredient
> 2. Delete ingredient


###### Section "Recipe book"

> User can:
> 1. View all recipes
> 2. Add new recipe
> 3. Edit recipe
> 4. Delete recipe


###### Section "Menu"

> User can:
> 1. Create daily menu in format: dish - portion amount.


###### Section "Shopping"

> User can:
> 1. Create shop list from menu.
